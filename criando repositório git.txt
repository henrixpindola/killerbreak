criando repositório git

rm -rf .git ( remover rastreios )

git init ( vou iniciar um novo repositório, do 0)

git add . (adicione todos os arquivos que tenho nesta pasta no novo repositório)

git commit -m "meu primeiro commit" (envia os arquivos pro repositório)

git branch -M main

git remote add origin https://github.com/usuário/repositóriocriado.git

git push -u origin main

git commit -m "A mudança" (realizar mudança no repositório, depois de criar, modificar, excluir arquivo e, fazer os ajustes no source control [+]. é um commit)

git push (empurrar a mudança)
